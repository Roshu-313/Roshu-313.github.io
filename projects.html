The problem
Business teams depend on developers to access data. This bottleneck slows decisions. Users can just ask: "Show top customers by revenue last month" — and get structured results with plain-English explanations, no SQL knowledge needed.
How it works
  • Step 1 → Schema-aware NL → SQL conversion using LangChain + OpenAI
  • Step 2 → Safe query execution with guardrails via SQLAlchemy
  • Step 3 → Structured results returned with data visualizations
  • Step 4 → Plain-English explanation of what the data means
Key features
Schema-aware NL → SQL Safe query execution Plain-English explanations Multi-database support User-friendly Streamlit UI
